# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-07
### Changed
- Major release with full modernization.
- Switch to Continuous Deployment on every push to `main`.
- Modernize project infrastructure (Makefile, GitHub Actions).
- Update Node.js dependencies to latest stable versions.
- Convert project to use ECMAScript Modules (ESM).
- Rename default branch from `master` to `main`.
- Adopt standard README header and development guide.
- Install and configure `pre-commit` hooks for local validation.

## [0.1.1] - 2026-04-07
### Changed
- Minor version bump for deployment testing.
